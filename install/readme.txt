For windows users:

First install python, then wxpython and numpy (you find the installers in this folder).
To launch simply click on Optical.

For the others:

Optical is a python script that needs python wxpython and numpy to be installed on your system.
You can download the right version for your OS from 

www.python.org
www.wxpython.org
numpy.scipy.org

After the above software has been installed you can directly run Optical.py.
